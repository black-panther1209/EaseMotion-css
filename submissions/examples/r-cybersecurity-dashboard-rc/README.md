# Cybersecurity Operations Dashboard Showcase

## What does this do?

Provides a cybersecurity-focused dashboard for monitoring threats, incidents, security alerts, and response metrics.

## Features

- Threat monitoring cards
- Security incident tracking
- Detection rate metrics
- Incident response progress
- Security activity feed
- Responsive dashboard layout

## How is it used?

Open `demo.html` directly in a browser.

## Why is it useful?

It demonstrates real-world SOC (Security Operations Center) dashboard patterns while showcasing modern animations, status indicators, and monitoring interfaces.
